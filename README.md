# LiftUpLabs 🚀

Welcome to the **LiftUpLabs** repository! This is a modern, enterprise-grade web application built to showcase technical domains, specialized services, and a premium interactive user experience.

## 🌟 Overview

LiftUpLabs is designed with high-fidelity aesthetics in mind, featuring:
- **Immersive User Interface:** Built with a custom design system focusing on premium modern web design, dynamic animations, and a unified brand identity.
- **Interactive Elements:** Features custom mouse cursors, scroll animations, micro-animations on interactive components, and high-quality UI elements.
- **Robust Architecture:** Powered by the latest web technologies for maximum performance and maintainability.

## 🛠️ Technology Stack

This project is built using modern cutting-edge technologies:

- **Framework:** [Next.js](https://nextjs.org/) (v16 Canary)
- **Library:** [React](https://react.dev/) (v19)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (v4) with global CSS custom aesthetics.
- **Animations:** [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/), and `tailwindcss-animate` for dynamic and fluid interface interactions.
- **UI Components:** [Radix UI](https://www.radix-ui.com/) primitives for accessible, high-quality, and unstyled building blocks.
- **Forms & Validation:** `react-hook-form` and `zod` for strict and robust form handling.
- **Additional Tools:** `lucide-react` for icons, `embla-carousel-react`, and `recharts` for data visualization.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm (or pnpm/yarn) installed on your local machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/devabokare/liftuplabs-main.git
   ```

2. Navigate into the project directory:
   ```bash
   cd liftuplabs-main
   ```

3. Install the project dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the local server address (usually `http://localhost:3000` or the custom IP specified in the terminal output) to see the application running.

## 📁 Project Structure

- `app/` - Next.js App Router containing pages, layouts, and API routes.
- `components/` - Reusable React components including the core UI primitives (`components/ui`) and complex sections (like `AboutSection`, `ContactSection`).
- `hooks/` - Custom React hooks for specialized logic.
- `lib/` - Utility functions and shared helpers.
- `public/` - Static assets such as images, icons, and fonts.
- `styles/` - Global stylesheets and Tailwind configurations.

## ✨ Features

- **About & Team Sections:** Interactive member profile cards revealing professional bios and capabilities.
- **Specialized Services:** Highlighting core domains of expertise such as ERP and Cloud solutions.
- **Contact Interface:** Clean, validated forms to seamlessly connect with the LiftUpLabs team.
- **Global Theming:** Sleek dark mode capabilities and custom scroll behaviors (using `lenis`).
